1. Descripción del Proyecto

El presente proyecto consiste en el diseño y desarrollo de un sitio web tipo Portfolio Personal bajo la arquitectura de "Single Page Application" (SPA). El objetivo principal es crear una identidad digital profesional que demuestre competencias técnicas en desarrollo web frontend, utilizando exclusivamente tecnologías nativas: HTML5, CSS3 y JavaScript (Vanilla JS), sin el uso de frameworks externos.

La aplicación destaca por su interfaz dinámica, que incluye un sistema de navegación sin recargas, un fondo animado con partículas interactivas mediante HTML5 Canvas, efectos de inclinación 3D y un sistema de gestión de temas (Claro, Oscuro y Hacker).

2. Justificación

La elección de este proyecto se justifica por la necesidad de consolidar los conocimientos adquiridos durante el curso en el ámbito de los Lenguajes de Marcas. Se ha optado por no utilizar librerías como Bootstrap o React para demostrar un dominio profundo de los fundamentos de la programación web, controlando manualmente el DOM, las animaciones y la gestión de estados.

3. Metodología

Para el desarrollo del proyecto se ha seguido una metodología iterativa e incremental:

  • Análisis y Diseño: Definición de secciones y estética "Cyber/Tech".
  • Estructuración (HTML): Esqueleto semántico con HTML5.
  • Estilizado (CSS): Metodología Mobile-first, Flexbox, Grid y Variables CSS.
  • Programación (JS): Lógica de interacción, enrutamiento simulado y Canvas.

4. Tecnologías y Retos

Sistema de Partículas (Canvas)

Sincronizar el movimiento del ratón con el redibujado de líneas entre puntos requirió cálculos matemáticos de distancia en cada frame de la animación.

Gestión de Temas

La lógica del botón de temas iterar sobre un array de tres opciones (Claro, Oscuro, Hacker) utilizando el operador módulo en JavaScript.

5. Conclusiones

Tras la finalización del proyecto, se concluye que es posible crear aplicaciones complejas tipo SPA sin necesidad de frameworks pesados, optimizando el rendimiento mediante CSS puro y utilizando JS solo para la lógica compleja. El resultado cumple con el objetivo de servir como carta de presentación laboral.
